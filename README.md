# Directors' Timeline Archive

現代映画を定義する巨匠たちの軌跡を解読するためのデジタルアーカイブです。

## 特徴
- **膨大な作品データ**: スタンリー・キューブリックからアリ・アスターまで、幅広い年代の監督を網羅。
- **比較機能**: 2人の監督を並べて表示し、同時代の歴史背景とともにそのキャリアを比較できます。
- **動的なタイムライン**: 選択した監督のデビュー年に合わせて年表の開始位置を自動調整します。
- **レスポンシブデザイン**: PC、タブレット、スマートフォンで快適に閲覧可能です。

## 技術スタック
- HTML5
- CSS3 (Vanilla CSS)
- JavaScript (Vanilla JS)
